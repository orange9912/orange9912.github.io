# 前言

趁着有空，把自己平时使用的一些函数抽离出来做个记录。

# 防抖

防抖的思想是：当某个事件处理函数被短时间内高频出发时，只有触发完一定时间没有再触发才会执行。

可以用于输入框监听等高频触发事件的地方。

# 节流

使用场景同防抖，只不过核心思想有些不一致。

节流是某个事件处理函数被短时间内高频触发时，一定时间内只执行一次，可以降低执行频率。
